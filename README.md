# Book Management REST API

A Flask-based REST API for managing a book inventory system with full CRUD operations.

## Tech Stack
- Python 3.x
- Flask
- Flask-SQLAlchemy
- MySQL
- PyMySQL

## Features
- ✅ Create new books
- ✅ Read book details (by ID or name)
- ✅ Update book information
- ✅ Delete books
- ✅ List all books

## Setup

### Prerequisites
- Python 3.x
- MySQL server
